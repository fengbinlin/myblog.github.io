算法-图-1

2022年2月1日21:51:43
四种重要的图模型：
无向图（简单连接）
有向图（连接有方向性）
加权图（连接带有权值）
加权有向图（连接既有方向性又带有权值）
图的定义：图是由一组顶点和一组能够将两个顶点相连的边组成的。
一般使用0至V-1来表示一张含有V个顶点的图中的各个顶点。
用V-W或者W-V表示连接他们的边。
特殊的图
自环：一条连接一个顶点和其自身的边
连接同一对顶点的两条边称为平行边。
多重图：含有平行边的图
简单图：没有平行边和自环的图
路径是由边顺序连接的一系列顶点。
简单路径：一条没有重复顶点的路径
环：一条至少含有一条边且起点和终点相同的路径。
简单环：不含有重复顶点和边的环。
路径/环的长度：其中包含的边数。

连通图：任意一个顶点都存在一条路径到达另一个任意顶点
非连通图由若干连通部分组成，他们都是极大连通子图
树：无环连通图
森林：互补相连的树组成的集合称为森林。
连通图的生成树：含有所有顶点且是一棵树

理解算法最好的方法是在一个简单的例子中跟踪它的行为。

2022-01-27 10:08:59

选择排序

A = [64, 25, 12, 22, 11] #指定一个数列

for i in range(len(A)): 
    

    min_idx = i #默认最小值的索引是第一个
    for j in range(i+1, len(A)): 
        if A[min_idx] > A[j]: #若在后面找到比第一个小的就替换
            min_idx = j 
                
    A[i], A[min_idx] = A[min_idx], A[i] #交换值

插入排序

A = [12, 11, 13, 5, 6]
for i in range(1, len(A)):
    key = A[i]
    j = i - 1
    while j >= 0 and key < A[j]:
        A[j + 1] = A[j]
        j -= 1
    A[j + 1] = key
print(A)

希尔排序（递减增量排序算法）

A=[12, 34, 54, 2, 3]
n = len(A)
gap = int(n/2)
while gap > 0: 

    for i in range(gap,n): 

        temp = A[i] 
        j = i 
        while  j >= gap and A[j-gap] >temp: 
            A[j] = A[j-gap] 
            j -= gap 
        A[j] = temp 
    gap = int(gap/2)
print(A)

希尔排序，我看了很多种解释，经过演绎才明白他的原理哈哈哈。它是插入排序的升级版。解决的问题是，插入排序只能一步步移动，移动得远的话就很不方便。

在我的理解下，希尔排序是隔一段相同的距离A便取一个数作为一个子数列，起点不同的话，就可以把原始要排列的数组拆分成多个子数列，分别对每个子数列使用插入排序。排玩之后，在不断缩小开始的距离A，这里的代码里面采用的方法是不断除以2.当最终A为1的时候其实就是最后再对多次操作后的数列进行一次插入排列。

我是看了这位大佬的文章明白的：https://zhuanlan.zhihu.com/p/409836908

归并排列

菜鸟教程的代码目标（自上向底，接下来再讲分类）
def merge(arr, l, m, r):
n1 = m - l + 1
n2 = r - m

    # 创建临时数组
    L = [0] * (n1)
    R = [0] * (n2)

    # 拷贝数据到临时数组 arrays L[] 和 R[] 
    for i in range(0, n1):
        L[i] = arr[l + i]

    for j in range(0, n2):
        R[j] = arr[m + 1 + j]

        # 归并临时数组到 arr[l..r]
    i = 0  # 初始化第一个子数组的索引
    j = 0  # 初始化第二个子数组的索引
    k = l  # 初始归并子数组的索引

    while i < n1 and j < n2:
        if L[i] <= R[j]:
            arr[k] = L[i]
            i += 1
        else:
            arr[k] = R[j]
            j += 1
        k += 1

    # 拷贝 L[] 的保留元素
    while i < n1:
        arr[k] = L[i]
        i += 1
        k += 1

    # 拷贝 R[] 的保留元素
    while j < n2:
        arr[k] = R[j]
        j += 1
        k += 1


def mergeSort(arr, l, r):
    if l < r:
        m = int((l + (r - 1)) / 2)

        mergeSort(arr, l, m)
        mergeSort(arr, m + 1, r)
        merge(arr, l, m, r)


arr = [12, 11, 13, 5, 6, 7]
n = len(arr)
print("给定的数组")
for i in range(n):

mergeSort(arr, 0, n - 1)

理解这个排序的算法也费了我好些功夫
再解释这个算法的思想和代码实现之前，先介绍一下，归并算法有三种：
原地归并
自上而底的归并
自底向上的归并

原地归并我还不会，时间关系……

自上向底的归并，思想上是把原序列不断二分到2个一组，然后组内归并，归并完之后合并为一个整体，和隔壁的另一个整体组成一个新的组，然后再归并，不断操作，最终得到的是两个排好序的序列，最后一次归并即可

自底向上的归并则反过来，两个两个凑成一组归并，归并完后看成一个整体，和隔壁的组成一组归并，最后回到顶层。

快速排序

这也是菜鸟教程的快速排序的目标：

def partition(arr, low, high):
    i = (low - 1)  # 最小元素索引
    pivot = arr[high]

    for j in range(low, high):

        # 当前元素小于或等于 pivot
        if arr[j] <= pivot:
            i = i + 1
            arr[i], arr[j] = arr[j], arr[i]

    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return (i + 1)


# arr[] --> 排序数组
# low  --> 起始索引
# high  --> 结束索引

# 快速排序函数
def quickSort(arr, low, high):
    if low < high:
        pi = partition(arr, low, high)

        quickSort(arr, low, pi - 1)
        quickSort(arr, pi + 1, high)


arr = [10, 7,234,4,12,42,45, 8, 9,4 ,51, 5]
n = len(arr)
quickSort(arr, 0, n - 1)
print("排序后的数组:")
for i in range(n):
    print("%d" % arr[i]),

这个快速排序也花了点功夫理解。

思想上就是选一个基准，把小于他的丢在左边，大于他的放在右边。不断递归

以上代码上的操作是把最靠右边的作为基准，接着比他小的放在最左边排着队，最后把最右边那个基准调到左边比他小的那群数的最右边挨着，接着下一步递归。

冒泡排序

还好这个简单一点，一下是菜鸟教程的代码模板：

def bubbleSort(arr):
    n = len(arr)

    # 遍历所有数组元素
    for i in range(n):

        # Last i elements are already in place
        for j in range(0, n-i-1):

            if arr[j] > arr[j+1] :
                arr[j], arr[j+1] = arr[j+1], arr[j]

arr = [64, 34, 25, 12, 22, 11, 90]

bubbleSort(arr)

print ("排序后的数组:")
for i in range(len(arr)):
    print ("%d" %arr[i]),

有一个技巧点提一下，就是冒泡比较的最右端不断通过n-i-1递减，因为前面冒泡操作已经排好了，右边的几个是确定排好的，不需要再比较了。
